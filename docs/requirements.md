# 메모 + 캘린더 앱 - 상세 요구사항 문서

**프로젝트명**: MemoCalendar (메모캘린더)
**작성일**: 2026-02-22
**버전**: 1.0.0

---

## 1. 프로젝트 개요

### 1.1 앱 소개
메모와 캘린더를 자연스럽게 결합한 생산성 앱으로, 날짜별로 메모를 관리하고 캘린더에서 메모를 한눈에 확인할 수 있습니다.

### 1.2 타겟 사용자
- 일정 관리와 메모를 동시에 사용하는 사람
- 날짜별로 아이디어, 할 일, 일기를 기록하는 사람
- 스마트한 정보 관리가 필요한 직장인, 학생

### 1.3 핵심 가치
- **통합**: 메모와 캘린더가 하나의 앱에서 자연스럽게 연동
- **효율성**: 빠른 작성, 검색, 분류로 정보에 빠르게 접근
- **유연성**: 다양한 뷰와 분류 방식으로 원하는 방식으로 관리

## 2. 기능 요구사항

### 2.1 메모 CRUD (핵심 기능)

#### 2.1.1 메모 생성
- **필수 항목**: 내용 (최소 1자 이상)
- **선택 항목**:
  - 제목 (최대 100자)
  - 내용 (마크다운 문법 지원, 최대 50,000자)
  - 색상 (기본 12가지 색상 팔레트)
  - 날짜 범위 (시작일 ~ 종료일)
  - 태그 (다중 선택)
  - 폴더 (단일 선택)
  - 고정(핀) 여부
  - 즐겨찾기 여부
  - 리마인더 시간 설정
  - 체크리스트 항목
- **자동 기록**: 생성 일시, 수정 일시

#### 2.1.2 메모 조회
- 리스트뷰: 제목, 내용 미리보기, 날짜 표시
- 그리드뷰(카드): 색상 카드 형태로 2열 표시
- 캘린더뷰: 달력에 메모 표시
- 고정(핀) 메모는 목록 상단에 표시
- 즐겨찾기 메모 별도 필터링

#### 2.1.3 메모 수정
- 모든 필드 수정 가능
- 마지막 수정 일시 자동 갱신

#### 2.1.4 메모 삭제
- 소프트 삭제 (휴지통으로 이동)
- 단일/다중 선택 삭제
- 30일 후 자동 영구 삭제

### 2.2 캘린더 연동

#### 2.2.1 날짜 연결
- 메모 생성/수정 시 날짜 범위 지정 가능
  - 시작일: 메모가 시작되는 날짜
  - 종료일: 메모가 끝나는 날짜 (단일 날짜 선택도 가능)
- 날짜 없는 메모도 지원

#### 2.2.2 캘린더 화면
- 월 단위 캘린더 표시
- 메모가 있는 날짜에 색상 도트 표시 (최대 3개)
- 날짜 범위 메모는 달력에 이벤트 바로 표시
- 날짜 클릭 시 해당 날짜의 메모 목록 표시
- 주간 뷰 / 월간 뷰 전환

### 2.3 폴더/태그 분류

#### 2.3.1 폴더
- 폴더 생성, 이름 변경, 삭제
- 폴더별 색상 지정
- 폴더 내 메모 개수 표시
- 기본 폴더: "전체", "개인", "업무", "아이디어"
- 폴더 삭제 시 메모는 "전체"로 이동

#### 2.3.2 태그
- 태그 생성, 삭제
- 태그별 색상 지정
- 메모에 다중 태그 부여
- 태그 클라우드 또는 목록 형태로 태그 관리
- 태그별 메모 필터링

### 2.4 검색 기능

#### 2.4.1 기본 검색
- 제목/내용 실시간 검색 (타이핑 즉시 결과 표시)
- 대소문자 구분 없이 검색
- 검색 결과에서 매칭 텍스트 하이라이트

#### 2.4.2 필터 검색
- 폴더별 필터
- 태그별 필터 (다중 태그 AND/OR 조건)
- 날짜 범위 필터
- 색상 필터
- 고정/즐겨찾기 필터

#### 2.4.3 검색 기록
- 최근 검색어 최대 10개 저장
- 검색 기록 개별/전체 삭제

### 2.5 리마인더

#### 2.5.1 알림 설정
- 메모별 알림 시간 설정 (날짜 + 시간)
- 반복 알림: 없음, 매일, 매주, 매월
- 알림 텍스트: 메모 제목 또는 커스텀 메시지

#### 2.5.2 알림 표시
- 로컬 푸시 알림
- 알림 탭 시 해당 메모로 이동

### 2.6 다양한 뷰

#### 2.6.1 메모 목록 뷰
- **리스트뷰**: 한 행에 하나의 메모, 제목 + 내용 미리보기 + 날짜
- **그리드뷰**: 2열 카드 형태, 색상 배경, 제목 + 내용 미리보기
- 뷰 전환 버튼 (AppBar 우측)
- 마지막 선택 뷰 저장

#### 2.6.2 캘린더 뷰
- 월간 캘린더에 메모 점(도트) 표시
- 날짜 선택 시 하단 패널에 메모 목록

### 2.7 휴지통

#### 2.7.1 삭제 관리
- 삭제된 메모 목록 표시
- 삭제 일시 표시
- 남은 자동 삭제 일수 표시 (예: "23일 후 영구 삭제")
- 개별/전체 복구
- 개별/전체 영구 삭제

### 2.8 설정

#### 2.8.1 테마 설정
- 라이트 모드 / 다크 모드 / 시스템 설정 따름
- 기본 메모 색상 설정

#### 2.8.2 뷰 설정
- 기본 뷰 설정 (리스트/그리드/캘린더)
- 정렬 기준 (수정 일시, 생성 일시, 제목, 날짜)
- 정렬 순서 (오름차순/내림차순)

#### 2.8.3 알림 설정
- 전체 알림 on/off

#### 2.8.4 데이터 관리
- 전체 데이터 초기화 (확인 다이얼로그 포함)
- 앱 정보 (버전, 오픈소스 라이선스)

### 2.9 체크리스트

#### 2.9.1 체크리스트 항목
- 메모 내 체크리스트 항목 추가/삭제
- 항목 체크/해제
- 항목 순서 드래그로 변경
- 완료된 항목 취소선 표시
- 완료율 표시 (예: "3/5 완료")

### 2.10 하단 네비게이션

#### 2.10.1 탭 구성
- 홈 (메모 목록)
- 캘린더 (캘린더 뷰)
- 검색
- 설정

## 3. 비기능 요구사항

### 3.1 성능
- 앱 초기 로딩: 2초 이내
- 검색 응답: 300ms 이내
- 메모 저장: 즉시 반영

### 3.2 저장소
- 로컬 저장소 사용 (SharedPreferences + 파일 시스템)
- 오프라인 완전 지원

### 3.3 플랫폼
- Android API 21+
- iOS 12.0+

### 3.4 UI/UX
- Material Design 3 가이드라인 준수
- 다크/라이트 모드 모두 지원
- 한국어 UI (기본 언어)
- 직관적인 제스처 (스와이프로 삭제, 길게 누르기로 다중 선택)

### 3.5 접근성
- 충분한 색상 대비
- 적절한 터치 영역 크기 (최소 48x48dp)
- 스크린 리더 지원 (Semantics)

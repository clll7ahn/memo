# MemoCalendar 사용 가이드

## 앱 소개

**MemoCalendar**는 메모와 캘린더를 결합한 Flutter 기반의 메모 앱입니다.
일상적인 메모 작성부터 일정 관리, 체크리스트, 알림(리마인더)까지 하나의 앱에서 관리할 수 있습니다.

### 주요 특징
- 메모 CRUD (생성, 조회, 수정, 삭제)
- 캘린더 연동 (날짜별 메모 확인)
- 폴더 및 태그로 메모 분류
- 검색 기능 (제목 및 내용)
- 리마인더 (알림 설정, 반복 알림)
- 소프트 삭제 및 휴지통 관리 (30일 후 영구 삭제)
- 라이트/다크 모드 지원
- 그리드/리스트/캘린더 뷰 전환

## 주요 기능 설명

### 1. 메모 CRUD
- **생성**: 홈 화면 우하단 `+` 버튼으로 새 메모 작성
- **조회**: 홈 화면에서 리스트/그리드 뷰로 메모 목록 확인
- **수정**: 메모 탭 후 편집 화면 진입
- **삭제**: 메모 스와이프 또는 삭제 버튼 (휴지통으로 이동)

### 2. 캘린더 연동
- 하단 탭바의 캘린더 아이콘을 탭하여 캘린더 화면 진입
- 날짜를 선택하면 해당 날짜의 메모 목록 표시
- 메모에 날짜를 설정하면 캘린더에 표시

### 3. 폴더 분류
- 사이드 메뉴 또는 폴더 탭에서 폴더 관리
- 메모 작성/편집 시 폴더 지정 가능
- 폴더별 색상 지정 지원

### 4. 태그 관리
- 메모에 여러 태그 추가 가능
- 태그별 색상 지정 지원
- 태그 화면에서 태그별 메모 필터링

### 5. 검색
- 상단 검색 아이콘 탭 또는 검색 탭 진입
- 메모 제목 및 내용 전체 검색
- 실시간 검색 결과 표시

### 6. 리마인더 (알림)
- 메모 편집 화면에서 알림 일시 설정
- 반복 알림: 없음 / 매일 / 매주 / 매월
- 알림 메시지 커스터마이징 가능

### 7. 휴지통
- 삭제된 메모는 30일간 휴지통에 보관
- 휴지통에서 메모 복원 가능
- 30일 경과 후 자동 영구 삭제
- 수동 영구 삭제 기능 제공

### 8. 설정
- 테마 설정: 라이트 / 다크 / 시스템 모드
- 기본 뷰 설정: 리스트 / 그리드 / 캘린더
- 정렬 기준: 수정 일시 / 생성 일시 / 제목 / 날짜
- 정렬 순서: 오름차순 / 내림차순
